# EduConnect - Educational Management System

## Overview
EduConnect is a comprehensive web-based platform designed for managing educational institutions. It provides role-based access control for different user types including Admin, HOD (Head of Department), Teachers, Students, and Managing Authority.

## Features
- User authentication and authorization with JWT
- Role-based access control (RBAC)
- Assignment management
- Attendance tracking
- Grade submission and management
- Messaging system
- Activity logging
- Department management
- Resource sharing

## Technology Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Other**: bcryptjs, multer, nodemailer

## Future Enhancements
- Face recognition attendance system using Python with OpenCV, LBPH, and Haar Cascade for real-time face detection and recognition
- MongoDB for student and attendance data management
- Tkinter for interactive GUI
- Supporting libraries: NumPy, Pandas, Pillow, and OpenPyXL for image processing and data handling

## Installation
1. Clone the repository
2. Navigate to the backend directory: `cd backend`
3. Install dependencies: `npm install`
4. Set up environment variables in `.env` file
5. Start the server: `npm start` or `npm run dev` for development

## Usage
After starting the server, access the application through the web browser at the designated port (typically 5001).
